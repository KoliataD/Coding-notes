Markdown Readme's

Markdown Readme's

Typically a readme will be written in Markdown which is why it is in my Markdown Notes
You want to make sure that in every project that you make that there is a readme file labeled readme.md with the following information in it applications name, mame of contributors, description of application, link to site on GitHub pages, a "technologies used" section, an application setup instructions, a "known bugs" page, a copyright, date, and liscense information. You want to make sure that your readme is in the top of your directories so that it is easy to find.

Markdown Tags, and elements

Markdown Tags, and elements

Hashtags
Hastags will change the size of your text the less hashtags before your text the larger it will be the smallest you can make it is 6 hastags ######. To use these you simply put it before your text. "#pineapple" This will make a large text pineapple "######pineapple" this will make a small text pineapple.

Paragraphs
Paragraphs will be automatically made with normal text.

Emphasizing text
To emphasize text simply surround it in underscores likes this _pineapple_ this will be a emphasized pineapple.

Unordered list
To make an unordered list type in an asterisk and than a space * like this this will turn into a list

Ordered list
To make an ordered list simply type in what you would for an ordered list. Numnbers and dots.

Bold
To make something bold in markdown simply put 2 asterisks on both ends **pineapple** this will be a bold pineapple.

Linking images and webpages
To make a link in markup we put the text we want to show up in brackets, and than without putting a space we put the link we want it to link to in parenthesis. [this will be the text that shows up](thiswillbealink.com). To make this into an image just put a exclamation mark infront of it.
