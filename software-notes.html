Dev Tools

Accessing google dev tools
To access dev tools we can typically go to the 3 dots or the settings icon go to more tools than dev tools. We can also hit ctrl+shift+i. on mac you type cmd+opt+j also windows+alt+j for windows.

Using google dev tools
Dev tools can be used to help debug or to just see what css elements we would like to change. After we are in the dev tools we will be shown code that is in javascript even if we are writing in something else. in the console area it will show any bugs that it detects, and will show us the resource that failed, the error code, and than the file that is failing, or the line of code. If it is failing to find a file it is typically because the file doesn't exist, or because we typed it in incorrectly. This is what it will look like

Failed to load resource=net:: ERR_FILE_NOT_FOUND css.file:1


in the elemnts tab we will see every one of our elements in the order they are typed in we will be able to expand and shrink them. When we select one our elements like body it will tell us what css declarations are effecting it in a tab below.
