Linking Elements

Linking elements can link different files to a page.

<img>
This will create a image on your webpage. This tag requires a value for src, and optionally alt. Src will be the image adress, and alt will be alternative text.

it will look something like this:

<img src="img/image.jpeg" alt="this is a image">

<a>
This is a anchor element.This can add a link to text within it. This tag requires a value for href(hyperlinkreference). The value you will give href is the link to where you want it to go. When referencing within a page and not to an external link or internal file you will link it to an id with a # infront of it.

it will look something like this:

<a href="whateveryourlinkis#this-goes-to-a-internal-id>this is the text it will be linked to</a>

Text Properties, and Values

id
This property can be given to something to allow it to be referenced. You can take the id you give something, and put it into a hypertext reference

it will look something like This

<h1 id="our-id">random text</h1>
<a href="#our-id">The text that will be the link</a>



class
this can be added to text or text groups in divisions or spans to make rules or effect large groups of text within the same class. Typically this is used for css styling.
