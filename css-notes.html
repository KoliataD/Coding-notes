Css File Setup

Creating Css Styling Sheets

Creating a Css stylesheet To create a Css sheet we typically first want to create a directory file called css, and than within that css directory we want to create our css stylesheets we must make sure that they end with .css so that they are read as style sheets. After this we need to link our style sheet to our page.

Linking Files

<head>
  <link href="css/(whatever your files name is" rel="stylesheet" type="css/stylesheet">
</head>
This is how you link your stylesheet to whatever webpage you are working on. Href stands for hypertext reference rel stands for relationship, and type tells our shell the type of stylesheet it is.

Css Basic Coding Principles

Css Terminology

Css stands for Cascading style sheet. Css allows us to add design to our webpages.

Declarations
A declaration is basically a rule, or a modifier we can add to large sections of similar code. To set declarations in css we use brackets {} this is what they look like. To make brackets into a declaration we simple put a element or a class name with a dot before it infront of it than give it properties set to the values we want them to be. It will look like this h1{color: blue;} or for a class .this-is-a-class{color: red ;} Notice that we don't use an equal sign we use a colon, and afterwards we use a semi-colon. The colon is basically our equal sign, and the semi colon tells us that their are more declarations to be read. when we have a lot of declarations together it is a declaration block.

Indentation, and spacing

Comments
to write a comment in css you write /*THIS IS A COMMENT*/ on windows, and mac their are commands to automatically write comments on windows it is ctr + / and on mac it is cmd + /

Indentation
When you are writing declarations in css your element and your first bracket will be on the same line, and than the following declarations will be indented once. every declaration will start from the beginning. It will look something like This

.class{
  color: blue;
  text-font: sans serif;
}

Css Properties, and Values

Text Properties, and Values

Color
This will change the color of your text

Text-align
This will change where your text shows up it can be left right or center.

font-style
This changes the style of font you have like italic or normal.

font-family
This is a much more specific font because their are lots of fonts the one we are currently using is sans-serif.

font-size
This changes the size of your text it is typically measured in pixels or px.

line-height
this will change the distance between your lines of text that you have.

font-weight
For this you can make it bold or normal. This will change the thickness of your text but not the overall size.
width
This will change the amount of space that text will take up horizontally this can be in px or percentages.

float
Changing this property will change where text or images show up on your page

media
Media is a property that will specify what resolution it works best with it can be low, and high.

Block Properties, and Values

Box properties, and values
to make a box we need 3 properties, border, padding, maring, and content. In the middle is your content/text around that is padding, around that is border, and lastly surrounding border is margin.
border
for the border we must set a px size, a color, and a type the type I usually use is solid.

padding
this will change the distance your text has to the border you can set a color, and a distance in pixels.

margin
The margin will change the distance it has to the edge of the page we can change this for the left side right side bottom, and top.
